Modular City Alley Pack

Use LINEAR COLOR SPACE to get best results and Deferred Rendering Path with built-in render pipeline.

When using directional light, to fix the light coming throug backside of the wall pieces, select all wallpieces and change
from Mesh renderer settings to Cast shadows two sided. You might also need to add some lightblocker objects.
